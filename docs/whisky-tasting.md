# ウィスキーテイスティングノート 知識ベース

## 1. テイスティングの基本プロセス

ウィスキーのテイスティングは以下の5段階で行う。

### Step 1: 外観（Appearance / Colour）

グラスを45度に傾け、白い背景に対して色を観察する。

| 色 | 意味 |
|---|---|
| 淡い黄金色（Pale Gold） | 若い熟成、バーボン樽 |
| 琥珀色（Amber） | 中程度の熟成 |
| 銅色・マホガニー（Copper / Mahogany） | シェリー樽、長期熟成 |
| 濃い茶色（Dark Brown） | 長期熟成または濃色シェリー樽 |

**レッグス（Legs）:** グラス内壁を流れる雫の速さと量で粘度・オイリーさを判断する。

### Step 2: 香り（Nose / Aroma）

人間の味覚の約80%は嗅覚由来。

1. グラスをゆっくり近づけ、口で息を吸いながら最初の印象を掴む
2. グラスのリムより少し下から徐々に深く鼻を入れる
3. 少量の水を加えて再度ノージング（香りが開く場合がある）

### Step 3: 味わい（Palate / Taste）

少量を口に含み、舌全体に広げて15〜30秒保持する。感じるべき要素：

- **マウスフィール（Mouthfeel）:** 粘度、とろみ、なめらかさ
- **アタック:** 最初に感じる強さ
- **展開:** フレーバーの変化・深まり方
- **バランス:** 甘み・辛み・酸・苦みの調和

### Step 4: フィニッシュ（Finish / Aftertaste）

飲み込んだ後の余韻の長さと質。

| 長さ | 目安 |
|---|---|
| Short（短い） | 15秒未満 |
| Medium（中程度） | 15〜45秒 |
| Long（長い） | 45秒〜2分 |
| Very Long / Endless | 2分以上 |

### Step 5: 加水・再評価（Optional）

少量の常温水（2〜3滴）を加えることでエチルエステルが分解され、異なるフレーバーが解放される。最適な飲用濃度は20〜30% ABV。

## 2. フレーバーホイール（SWRI ペントランド・フレーバーホイール）

1970年代にスコッチウィスキー研究所（SWRI）が開発した業界標準。8大カテゴリーで構成され、発酵・蒸溜由来（1〜6）と熟成由来（7〜8）に大別される。

### カテゴリー1: Cereal（シリアル/穀物）

| サブカテゴリー | 具体例 |
|---|---|
| Cooked Mash | お粥、ライスプディング |
| Cooked Veg | 茹でキャベツ、コーン |
| Husky | 麦わら、乾いた穀物 |
| Malt Extract | ビスケット、麦芽糖 |
| Yeasty | パン、ビール酵母 |

### カテゴリー2: Fruity（フルーティ/果実）

| サブカテゴリー | 具体例 |
|---|---|
| Citric（柑橘） | オレンジ、レモン、グレープフルーツ |
| Fresh Fruit（生果実） | リンゴ、洋梨、桃、アプリコット |
| Cooked Fruit（加熱果実） | 煮りんご、マーマレード、ジャム |
| Dried Fruit（ドライフルーツ） | レーズン、イチジク、プルーン、フルーツケーキ |
| Solvent（溶剤様） | エステル過剰時のネイルリムーバー臭 |

### カテゴリー3: Floral（フローラル/花・草）

| サブカテゴリー | 具体例 |
|---|---|
| Fragrant（芳香） | ラベンダー、ローズウォーター、香水 |
| Greenhouse（温室） | ゼラニウム、草花 |
| Leafy（葉っぱ） | 青葉、刈り草、松の実 |
| Hay（干し草） | 干し草、ヘザー、ハーブ、セージ |

### カテゴリー4: Peaty（ピーティ/泥炭）

| サブカテゴリー | 具体例 |
|---|---|
| Medicinal（薬品） | ヨード、消毒液、絆創膏 |
| Smoky（煙） | キャンプファイヤー、燻製、コールタール |
| Kippery（燻製魚） | 燻製ニシン、スモークサーモン |
| Mossy（苔） | 湿地、泥炭地、アース |

### カテゴリー5: Feinty（フェインティ）

| サブカテゴリー | 具体例 |
|---|---|
| Honey（蜂蜜） | ハチミツ、ゴールデンシロップ、蜜蝋 |
| Leathery（革） | 古い革、なめし皮 |
| Tobacco（タバコ） | 葉巻、たばこ |
| Sweat & Plastic | 汗、ゴム |

### カテゴリー6: Sulphury（スルフリー/硫黄）

| サブカテゴリー | 具体例 |
|---|---|
| Coal Gas | マッチ、硫黄 |
| Rubbery | タイヤ、ゴム |
| Vegetative | キャベツ、たまねぎ（硫黄化合物由来） |

### カテゴリー7: Woody（ウッディ/木）※熟成由来

| サブカテゴリー | 具体例 |
|---|---|
| Toasted（焦げ） | トースト、コーヒーグラウンド、リコリス |
| Vanilla/Custard | カスタード、スポンジケーキ、タフィー |
| Old Wood（古い木） | カビ、段ボール、鉛筆、インク |
| New Wood（新しい木） | 樹脂、シガーボックス、生姜、胡椒 |

### カテゴリー8: Winey（ワイニー/ワイン）※熟成由来

| サブカテゴリー | 具体例 |
|---|---|
| Sherried（シェリー） | ワイン、ポート、マデイラ |
| Nutty（ナッツ） | クルミ、ヘーゼルナッツ、アーモンド |
| Chocolate（チョコレート） | クリームチョコ、カカオ、ビターチョコ |
| Oily（オイリー） | 蝋燭、オリーブオイル |

| Sweet（甘み） | ハチミツ、バニラ、キャラメル、タフィー |
| Fruity（果実味） | リンゴ、洋梨、柑橘、ベリー類 |
| Spicy（スパイシー） | 黒胡椒、シナモン、ジンジャー、クローブ |

## 5. 産地別の特徴

### スコッチウィスキー（Scotch Whisky）— 5法定産地

| 産地 | 主な特徴 | 代表的フレーバー |
|---|---|---|
| **スペイサイド** | フルーティ、エレガント、甘い | リンゴ、洋梨、蜂蜜、シェリー、チョコレート |
| **ハイランド** | 多様、中程度のボディ | ヘザー、蜂蜜、フルーツ、時にスモーキー |
| **アイラ** | 強いピート、ヨード、海岸風 | スモーク、ヨード、海藻、燻製魚、塩 |
| **ローランド** | 軽い、ソフト、トリプル蒸溜 | 草、穀物、フローラル、クリーミー |
| **キャンベルタウン** | 塩気、オイリー、ブライニー | 海塩、トフィー、バニラ、軽いスモーク |
| **アイランズ** | スペイサイドとアイラの中間 | 海塩、ヘザー、スパイス、マイルドなスモーク |

### バーボン（Bourbon）

- **法的要件:** 新品のチャーアメリカンホワイトオーク樽のみ、コーン51%以上
- **主要フレーバー:** バニラ、キャラメル、バタースコッチ、コーン、焦がし樽
- **サブスタイル:** ライウィスキー（スパイシー）、ホイートウィスキー（ソフト）

### アイリッシュウィスキー（Irish Whiskey）

- **特徴:** トリプル蒸溜による滑らかさ、未発芽大麦の使用
- **主要フレーバー:** ハチミツ、バニラ、軽い果実、クリーミー

### ジャパニーズウィスキー（Japanese Whisky）

- **特徴:** スコッチの技術を継承しつつ、繊細さと調和（和）を重視
- **主要フレーバー:** 繊細なフルーツ、フローラル、軽いスモーク
- **ミズナラ樽:** サンダルウッド、お香、バニラ、トンカビーン

### その他

| 産地 | 特徴 |
|---|---|
| カナディアン | 軽く、スパイシー（ライ由来）、ブレンド中心 |
| インディアン | 温暖気候による急速熟成、スパイシー |
| タスマニアン | フルーティ、クリーン、個性的 |

## 6. アルコール度数と加水の影響

| ABV | 特徴 |
|---|---|
| 40% | EU法定最低値（スコッチ）、フレーバーは穏やか |
| 43〜46% | バランスが良く、香りが開きやすい |
| 46%+ | ノンチルフィルタード推奨域、油性化合物が保持される |
| 52〜66% | カスクストレングス、全フレーバーがフルに保存 |

### 加水の効果

- **化学的効果:** エチルエステル（フレーバー化合物）の分解・解放
- **推奨加水量:** 2〜3滴から始め、好みの度数（理想は20〜30%）まで
- **注意:** 氷や冷水は香りを閉じ込めるため推奨されない

## 7. グラスの選び方

| グラス | 特徴 | 用途 |
|---|---|---|
| **グレンケアン（Glencairn）** | チューリップ型、重い底、幅広ボウル→細いリム | テイスティング標準、香りを集約しやすい |
| **コピタ（Copita）** | 細長いステム、小さな開口部 | 高度数・繊細なウィスキー向け、手の熱が伝わりにくい |
| **チューリップ型グラス** | コピタに似た形状 | プロのブラインドテイスティングに使用 |
| **タンブラー/ロックグラス** | 幅広、底厚 | カジュアル飲用（テイスティングには不向き） |

## 8. プロの評価システム

### Jim Murray's Whisky Bible（最も広く知られる）

100点満点、4項目各25点。

| 評価項目 | 内容 |
|---|---|
| Nose（香り、25点） | 複数回スニッフィングで得られる香り |
| Taste（味わい、25点） | フレーバーの現れ方とリズム |
| Finish（フィニッシュ、25点） | 熟成の質を示す余韻の消え方 |
| Balance（バランス、25点） | 複雑さと調和 |

**スコアの目安:**

| スコア | 評価 |
|---|---|
| 80点以上 | 試す価値あり |
| 85点以上 | 優れた |
| 90点以上 | 卓越した |
| 95点以上 | 世界クラスの傑作 |

## 9. テイスティングアプリに必要な機能

### コア機能

| 機能 | 詳細 |
|---|---|
| テイスティングノート記録 | 色・香り・味・フィニッシュ・総合評価を構造化して記録 |
| 評価システム | 100点満点またはスター5つ形式 |
| ボトルデータベース | 銘柄情報（産地・蒸溜所・ヴィンテージ） |
| コレクション管理 | 所有ボトルのリスト、数量・飲み残し量の管理 |
| ウィッシュリスト | 欲しいボトルのリスト化 |

### 重要機能

| 機能 | 詳細 |
|---|---|
| フレーバーホイール/フレーバータグ | ビジュアルなフレーバー選択UI |
| フレーバーグラフ | レーダーチャートで香味のバランス表示 |
| 産地別フィルタリング | スコッチ/バーボン/ジャパニーズなど地域別検索 |
| コミュニティ機能 | 他ユーザーのレビュー閲覧 |

### 差別化機能

| 機能 | 詳細 |
|---|---|
| テイスティングフライト管理 | 複数ボトルの比較テイスティングセッション記録 |
| 写真添付 | ボトル・グラスの写真をノートに添付 |
| ブラインドテイスティングモード | 銘柄を隠してのテイスティング練習 |
| データエクスポート | CSV形式でのデータ書き出し |

### UI/UX 設計上の重要考慮点

- **初心者向けガイド付きフロー:** 段階的に誘導する入力UI
- **フレーバーホイールのタップ選択:** 複雑な用語を視覚的に選べるUI
- **オフライン対応:** セラーや外出先でも使えること（PWAのキモ）
- **シンプルモード vs 詳細モード:** 用途に応じた切り替え
